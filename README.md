# 統計検定道場 (Statistics Exam Preparation App)

統計検定 準1級・2級・1級の学習を支援する Web アプリケーションです。教科書（TeX 数式対応）、問題演習、模擬試験、学習進捗管理の機能を備えています。

## 機能一覧

| 機能 | 概要 |
|------|------|
| 教科書 | 23 トピックの講義コンテンツ。KaTeX による数式レンダリング、例題の解答折りたたみ表示 |
| 問題演習 | 184 問（準1級）の選択式・記述式問題。トピック別フィルタ、解説付き |
| 模擬試験 | タイマー付き模擬試験。自動採点と結果レビュー |
| 学習進捗 | トピック別正答率、回答履歴、連続学習日数（ストリーク） |
| ユーザー管理 | ログイン / 新規登録 / ゲストモード。学習データの永続化 |
| AI チャット | 統計学の用語をキーワードベースで質問可能（オフライン動作） |

## 技術スタック

- **フロントエンド**: React 19 + Vite 6
- **スタイリング**: Tailwind CSS 3
- **数式レンダリング**: KaTeX
- **アイコン**: Lucide React
- **データ永続化**: localStorage（ブラウザ内保存）

## セキュリティに関する注意事項

### 現在の実装（localStorage ベース）

本アプリのユーザー管理は **localStorage** を使用しており、以下の制約があります：

| 項目 | 状態 | 説明 |
|------|------|------|
| パスワード保存 | **平文** | localStorage にパスワードが平文で保存されます。本番環境ではバックエンドでのハッシュ化（bcrypt 等）が必須です |
| 認証方式 | **クライアントサイドのみ** | サーバーサイド認証はありません。localStorage の値を操作すれば認証をバイパスできます |
| データ保存先 | **ブラウザ内** | 学習データはブラウザの localStorage に保存されます。ブラウザのデータ消去で失われます |
| HTTPS | **未設定** | デプロイ時は必ず HTTPS を使用してください |
| CSRF/XSS | **基本対策のみ** | React の JSX エスケープによる基本的な XSS 防御のみ。ユーザー入力を `dangerouslySetInnerHTML` に渡す箇所はありません |

### 本番運用に向けた改善推奨事項

1. **バックエンド導入**: Firebase Auth、Supabase、または自前の API サーバーによるサーバーサイド認証
2. **パスワードのハッシュ化**: bcrypt / argon2 によるハッシュ化（サーバー側で実施）
3. **HTTPS 必須化**: Vercel / Netlify 等のホスティングサービスでは自動対応
4. **データベース導入**: 学習データを PostgreSQL / Firebase Firestore 等に永続化し、デバイス間同期を実現
5. **入力バリデーション強化**: メールアドレス形式チェック、パスワード強度チェック
6. **レート制限**: ログイン試行回数の制限（ブルートフォース対策）
7. **セッション管理**: JWT またはセッショントークンによる有効期限管理
8. **CSP ヘッダー**: Content-Security-Policy の設定で XSS リスクを低減

### 現時点での利用推奨範囲

- 個人学習用途（ローカル環境・個人デプロイ）: **問題なし**
- 少人数での限定共有: **ゲストモード推奨**（パスワード機能は使わない）
- 不特定多数への公開: **バックエンド導入後に実施すべき**

`vite.config.js` に `base` オプションの追加が必要：

```js
export default defineConfig({
  plugins: [react()],
  base: '/リポジトリ名/',
});
```

## 教科書コンテンツの記法

トピックファイル内で使用できる記法：

| 記法 | 用途 | 例 |
|------|------|-----|
| `$...$` | インライン数式 | `$\bar{x}$` |
| `$$...$$` | ディスプレイ数式 | `$$\sum_{i=1}^n x_i$$` |
| `**...**` | 太字見出し | `**定義**` |
| `- ...` | 箇条書き | `- 項目1` |
| `【例題】` | 問題マーカー | 青いボックスで表示 |
| `【解答】...【/解答】` | 折りたたみ解答 | デフォルト非表示 |
